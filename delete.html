ACCOUNT / DATA DELETION - App "ReLife"

Last updated: 2026-01-26

You can request deletion of server-side data stored by us (e.g., restore/entitlement data for in-app purchases) as follows:

1) In the app (recommended)

Open in the app:

Settings -> Privacy -> Delete account/data

There you can trigger deletion directly.

2) Without the app (web request)

If you no longer have the app installed, you can request deletion here:

Please send an email to: info@reimgen.com

Subject: "Deletion request ReLife"

If available, include a restore/user ID shown in the app:

Settings -> Privacy -> Restore ID

If you do not have a restore/user ID, please provide suitable proof so we can match your request (e.g., transaction ID / order ID from the store receipt). We use this information solely to prevent abuse and process your deletion request correctly.

Which data are deleted?

  • Server-side restore/entitlement data (mapping "user ID <-> purchase status"),
  • related technical log data, unless required longer for security or legal reasons.

What is not deleted?

  • Data stored only locally on your device (e.g., save games) can be deleted in the app or by uninstalling.
  • Data we must retain for legal reasons (e.g., to fulfill legal obligations) may be blocked and deleted after retention periods expire.

Processing time

We process deletion requests as quickly as possible and confirm deletion or status via reply.
